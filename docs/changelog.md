# Changelog

All notable changes to Django ERD Generator.

## [Unreleased]

### Added
- MkDocs documentation site
- GitHub Pages deployment workflow

## [0.1.7] - 2026-04-23

### Added
- GIS field support (PointField, PolygonField, etc.)
- Data dictionary generation
- Multiple ERD dialects (Mermaid, PlantUML, dbdiagram)

### Changed
- Migrated from black/flake8 to ruff
- Improved relationship detection

### Fixed
- Bug fixes in field type mapping
- Documentation corrections

## [0.1.6] - 2026-04-20

### Added
- Initial release
- Basic ERD generation
- Mermaid.js support

## [0.1.5] - 2026-04-18

### Added
- PlantUML dialect support
- dbdiagram.io dialect support

### Changed
- Improved output formatting

## [0.1.0] - 2026-04-15

### Added
- Project scaffolding
- Basic model parsing
